{
  "files": [

Guidelines for libraries:
- For JavaScript/TypeScript: Include package.json with build tools (Rollup/esbuild)
- For Rust: Include Cargo.toml with lib configuration
- For Python: Include pyproject.toml or setup.py
- Create src/ or lib/ directory with main library code
- Add tsconfig.json or equivalent for the language
- Include .gitignore appropriate for the language
- Add comprehensive README.md with API documentation
- Include LICENSE file (MIT or Apache-2.0 recommended)
- Add CHANGELOG.md for version history
- Include tests/ directory with test examples
- Add examples/ directory with usage examples
- Create proper package exports and entry points
- Include TypeScript declaration files if applicable
- Add CI/CD configuration (.github/workflows for GitHub Actions)

The library should be production-ready and follow semver.
